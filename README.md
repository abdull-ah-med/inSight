# **inSight - AI-Powered Parking Management System**

## **Description**

inSight is an AI-powered parking management system developed to streamline vehicle entry, exit, and access control in parking facilities. This university project leverages License Plate Recognition (LPR) technology to automate check-in and check-out processes, improve security, and minimize the need for manual vehicle management. Developed using the MERN stack, inSight provides an accessible platform for both general users and administrators, featuring a responsive web application that enables easy user registration, real-time alerts, and admin-controlled access management.

## **Features**
- **User Authentication and Role Management**: Secure login and registration for users and admins.
- **License Plate Recognition (LPR)**: Automates vehicle check-in/check-out by recognizing license plates using AI.
- **User Dashboard**: Allows users to register vehicles, view entry/exit history, and receive real-time notifications.
- **Admin Dashboard**: Provides admins with tools to approve/deny registrations, manage access, view parking logs, and oversee flagged vehicles.
- **Real-Time Notifications**: Alerts users about their vehicle status and notifies admins of any unauthorized attempts.

## **Project Setup**

This project is a web application built with the MERN stack:
- **MongoDB** for database management
- **Express.js** as the server framework
- **React.js** for the frontend
- **Node.js** for backend development

## **Technologies Used**

- **License Plate Recognition API**: Enables automatic check-in and check-out functionality
- **MERN Stack**: Streamlined development with MongoDB, Express.js, React, and Node.js
- **WebSockets (Socket.IO)** for real-time notifications

---

## **Features Overview**

1. **User Features**:
   - Register/Login and view dashboard
   - Register vehicles (permanent and temporary)
   - View vehicle check-in/check-out history
   - Receive real-time notifications upon vehicle entry/exit

2. **Admin Features**:
   - Manage user and vehicle registrations
   - View all parking activity logs
   - Blacklist vehicles and manage flagged entries
   - Approve or reject vehicle access requests in real time
